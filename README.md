## Some packages used in this project
- Next.js
- React
- React Query
- Tailwind CSS
- Daisy UI
- Framer Motion
- Cypress

## Getting Started

Install

```bash
npm install
```

Run development server at [http://localhost:3000](http://localhost:3000)

```bash
npm run dev
```

Build

```bash
npm run build
```

Serve built application

```bash
npm run start
```

Test

```bash
npm run test
# or
npm run test:coverage # with coverage
# or
npm run test:open # Test in GUI
```
